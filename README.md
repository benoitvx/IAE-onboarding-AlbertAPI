# Onboarding Albert API

Notebook de découverte de l'API Albert (OpenGateLLM).

## Contenu

- Connexion à l'API Albert
- Génération de texte 
- System prompts
- Embeddings
- Mini RAG 

## Prérequis

1. Demander une clé API sur https://albert.sites.beta.gouv.fr
2. Définir la variable d'environnement :
```bash
   export ALBERT_API_KEY="ta-cle"
```

## Lancer le notebook

Ouvrir `notebook.ipynb` dans :
- Google Colab
- VS Code (extension Jupyter)
- JupyterLab
